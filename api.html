icdm2025.methods.ecme.ECME_algorithm(X_obs: ndarray, idx_t: int, amat: ndarray, Sigma_init: ndarray, n: int, tol: float = 1e-06, max_iter: int = 9000, verbose_every: int = 50)[source]¶

ECME for single fully-missing node T. - E-step: compute Q_hat = E[XX^T | X_O, Sigma_hat] - CM-step (complete-data): (A, Delta) <- fitdag(amat, s=Q_hat) - CE-step (observed-data): optimize Delta_t to maximize observed-data loglik over O

Returns: dag_fit (last CM-step), Sigma_hat, iters, converged

icdm2025.methods.ecme.E_step(X_obs: ndarray, idx_t: int, Sigma_hat: ndarray) → ndarray[source]¶

X_obs: n x (p-1) observed data (all variables except T) idx_t: index of the missing node T in 0..p-1 Sigma_hat: current full p x p covariance Returns Q_hat = E[XX^T | X_O] averaged over samples (p x p)

icdm2025.methods.ecme.ecme_update_delta_t_observed_ll(A: ndarray, Delta: ndarray, idx_t: int, S_oo: ndarray, O: list[int], jitter: float = 1e-09) → float[source]¶

ECME ‘Either’ step: with A, Delta_{-t} fixed, update Delta_t by maximizing the observed-data Gaussian log-likelihood for the observed block O.

Closed-form solution using Sherman–Morrison:

Let b = (A^{-1})_{O, t}, current Sigma_oo = S0 + Delta_t * b b^T, S0 := Sigma_oo - Delta_t * b b^T. Define u = S0^{-1} b, c = b^T u, d = u^T S_oo u. Then Delta_t_new = max(eps, (d - c) / c^2).

Returns the new (positive) Delta_t.

icdm2025.methods.px_em.PX_EM_algorithm(X_obs: ndarray, idx_t: int, amat: ndarray, Sigma_init: ndarray, n: int, tol: float = 1e-06, max_iter: int = 9000, verbose_every: int = 50)[source]¶

PX-EM for one fully-missing node T.

Iteration:

  1. E-step: Q = E[XX^T | X_O, Σ]

  2. Provisional M-step on Q -> (A,Δ)

  3. Choose expansion α via observed-likelihood geometry (rank-one, closed form)

  4. Expand Q -> Q’ with α on the T coordinate

  5. M-step on Q’ -> Shat’ (covariance in expanded coords)

  6. Reduce: Σ_new = G^{-1} Shat’ G^{-1}

icdm2025.methods.px_em.choose_alpha_px(A: ndarray, Delta: ndarray, idx_t: int, S_oo: ndarray, O: list[int]) → float[source]¶

Use the same rank-one geometry as ECME to pick a good α. Let Σ_oo = S0 + Δ_t b b^T where b = (A^{-1})_{O,t}, S0 = Σ_oo - Δ_t b b^T. Define u = S0^{-1} b, c = b^T u, d = u^T S_oo u. The observed-likelihood-optimal Δ_t is Δ_t* = (d - c) / c^2 (if positive). We pick α = sqrt( clamp(Δ_t* / Δ_t, α_min^2, α_max^2) ) so the expanded M-step tends to achieve the target variance along the b-direction.

Apply expansion X’ = G X with G diagonal (α at index t) to complete-data moments Q.

Scales row t and column t of Q by α (i.e., Q’ = D Q D with D=diag(…, α at t, …)).

Parameters:

  • Q – Square moment matrix (p×p).

  • idx_t – Index of target variable t (0-based).

  • alpha – Expansion factor applied at index t.

Returns:

Expanded moment matrix Q’.

icdm2025.methods.kiiveri_em.lmfit(s: ndarray, y: int, x: list[int], z: list[int] | None = None) → ndarray[source]¶

Ordinary least squares for a DAG node.

Fits a regression of node y on its parent indices x using the covariance matrix s (shape p×p). Optionally accepts a list z of fixed/indicator positions to mark (set to 1.0 in the output vector).

Parameters:

  • s – Covariance matrix (p × p), symmetric.

  • y – Index of the child node.

  • x – List of parent indices for node y.

  • z – Optional list of additional indices to mark as fixed (set to 1.0).

Returns:

Length-p vector m where:

  • m[parent] = beta for each parent in x

  • m[y] is the residual variance

  • m[z] = 1.0 for any provided fixed indices (if any)

Return type:

np.ndarray
